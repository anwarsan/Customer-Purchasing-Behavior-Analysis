# Customer Purchasing Behavior Analysis
Behavioral insights to improve conversion and transaction value

## Key Aspects of the Project
- Conversion rate (CVR) analysis
- Basket amount and spending behavior analysis
- Customer segmentation using RFM metrics
- Marketing effectiveness evaluation

## Methodology
- Exploratory Data Analysis (EDA)
- Correlation and regression analysis
- Logistic regression for purchase probability
- RFM-based customer segmentation

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Stakeholder-Centric Analysis
- Supports marketing and growth teams
- Identifies friction points in the user journey
- Enables data-driven retention strategies

## Ethical Considerations
- Privacy-conscious handling of customer behavior data
- Avoidance of manipulative marketing practices

## Deliverables
- Behavioral analysis notebooks
- Customer segmentation insights
- Strategic marketing recommendations
